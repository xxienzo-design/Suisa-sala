Política de Privacidade - SUISA SALAS

Última atualização: 02/07/2026

1. Informações coletadas

O bot SUISA SALAS pode coletar apenas as informações necessárias para seu funcionamento, como IDs de usuários, IDs de servidores, IDs de canais, IDs de cargos e configurações definidas pelos administradores.

2. Como utilizamos os dados

Os dados são utilizados exclusivamente para fornecer as funcionalidades do bot, armazenar configurações e melhorar a experiência dos usuários.

3. Compartilhamento de dados

O SUISA SALAS não vende, aluga ou compartilha dados pessoais com terceiros, exceto quando exigido por lei.

4. Segurança

Adotamos medidas razoáveis para proteger as informações armazenadas contra acessos não autorizados.

5. Retenção de dados

Os dados permanecerão armazenados enquanto forem necessários para o funcionamento do bot ou até que sejam removidos mediante solicitação ou exclusão do servidor.

6. Direitos dos usuários

Os usuários podem solicitar a exclusão de seus dados ou esclarecer dúvidas entrando em contato pelo servidor oficial de suporte.

7. Alterações nesta política

Esta Política de Privacidade poderá ser alterada a qualquer momento. A versão mais recente será sempre disponibilizada nesta página.

8. Contato

Para dúvidas relacionadas à privacidade, utilize o servidor oficial de suporte do SUISA SALAS.
